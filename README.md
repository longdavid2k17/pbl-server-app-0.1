# pbl-server-app-0.1
Server application for generating and managing data for machine learning

# Stack
- Java 11
- Spring Boot, Spring Security
- H2 Database / JPA
- Thymeleaf + HTML/CSS/JS/Bootstrap
- Javax Validation
- RabbitMQ
- Apache Commons CSV, Lombok
